The one with the CSS positioning cheatsheet


Fixed: A tag with fixed position takes selected tag and places it relative to the browser in a fixed position (stays put even when scrolling)

Absolute: A tag with an absolute position will position itself to the nearest relative with a position value other than static.

Relative: A tag with a relative position will position itself relative to its natural position, not the browser.

Static (default): This tag does not need to be called out specifically since this is the default /natural position of the item.

Margin: is the area outside of the border

Padding: is the area between the content and the border

Border: is the area between padding and margin

Inline: will condense the divs/tables to the bare minimum size based on the content (content size is left untouched) and also place them ‘in line’ or in a row fashion

Block: is an element that takes up the full width available, and has a line break before and after it.

Inline block: it will expand the padding around the content to fit the full potential of a block and then place each into a row

Bottom: a value for this property places item relative to the bottom of the browser

Right: a value for this property places item relative to the right of the browser

Top: a value for this property places item relative to the top of the browser

Left: a value for this property places item relative to the left of the browser
